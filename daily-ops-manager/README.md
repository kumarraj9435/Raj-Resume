# Daily Ops Manager — Amazon & Blinkit

> **Free, Self-Hosted** Daily Operations Manager for Amazon Sellers & Blinkit Vendors

## Features

| Module | Description |
|--------|-------------|
| Task Manager | Time-based daily schedule, priority tracking, Start/Done/Skip |
| Sales Analytics | Amazon + Blinkit sales, orders, revenue |
| Inventory Tracker | Stock levels, FBA/warehouse, reorder & aged alerts |
| PPC Ads Monitor | Campaign ACOS, spend tracking, pause/resume |
| Review Alerts | Rating monitor, auto-flag negative reviews |
| P&L Dashboard | Revenue, COGS, fees, profit & margin |
| Performance Score | Daily %, grade system (A+ to D), 30-day trends |
| Automation | Custom rules, notifications, daily task reset |

## Quick Start

```bash
cd daily-ops-manager
npm run setup     # Install all dependencies
npm run dev       # Start app (server + client)
```

Open http://localhost:3000

## Tech Stack (All Free)

- React 18 (Frontend)
- Express.js (Backend)
- SQLite (Database - zero config, file-based)
- Node-cron (Scheduled automations)

## Commands

```bash
npm run dev     # Development mode
npm run server  # Backend only (port 5000)
npm run client  # Frontend only (port 3000)
npm run build   # Production build
npm start       # Run production
```
